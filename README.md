# 🍽️ Zomato Data Analysis Using Python

## 📌 Project Overview
This project focuses on analyzing Zomato restaurant data to understand customer preferences, restaurant trends, and pricing patterns. The goal is to extract meaningful insights from the dataset using Python-based data analysis techniques.

The analysis helps identify popular cuisines, cost distribution, restaurant ratings, and location-based restaurant trends.

---

## 🎯 Objectives

- Clean and preprocess the Zomato dataset
- Analyze restaurant ratings and customer preferences
- Identify popular cuisines and restaurant types
- Study price distribution across restaurants
- Extract insights using data analysis techniques

---

## 🛠️ Tools & Technologies Used

- Python  
- Pandas – Data cleaning and manipulation  
- NumPy – Numerical analysis  
- Matplotlib – Data visualization  
- Seaborn – Statistical visualization  
- Jupyter Notebook – Development environment  

---

## 📂 Dataset Description

The dataset includes information such as:

- Restaurant Name
- Location / City
- Cuisines offered
- Average cost for two people
- Online delivery availability
- Table booking availability
- Customer ratings
- Number of votes

---

## 🔎 Project Workflow

### 1️⃣ Data Cleaning & Preprocessing
- Removed duplicate records
- Handled missing values
- Cleaned inconsistent data entries
- Converted data types for proper analysis

### 2️⃣ Exploratory Data Analysis (EDA)
- Analyzed distribution of restaurant ratings
- Identified most popular cuisines
- Examined cost distribution across restaurants
- Studied restaurant availability across different locations

### 3️⃣ Data Analysis
- Compared ratings across different cuisines
- Evaluated impact of online delivery on restaurant popularity
- Analyzed relationship between cost and customer ratings
- Identified top-rated restaurants

### 4️⃣ Data Visualization
- Bar charts for cuisine popularity
- Count plots for restaurant types
- Distribution plots for restaurant ratings
- Heatmaps for correlation analysis

---

## 📊 Key Insights

- Certain cuisines dominate restaurant listings
- Restaurants offering **online delivery** attract more customers
- Moderate pricing restaurants receive higher customer engagement
- Popular locations have higher restaurant density

---

## 🚀 Business Impact

This analysis helps food delivery platforms and restaurants:

- Understand customer preferences
- Optimize pricing strategies
- Identify high-demand cuisines
- Improve restaurant recommendations
